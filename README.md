# RN Concierge Aesthetics Website

This is a static website built with real provided assets. No AI-generated patient images are included.

## Preview locally
Open `index.html` in a browser.

## Deploy from GitHub to Netlify
1. Create a GitHub repository and push this folder.
2. In Netlify, choose "Add new site" > "Import an existing project".
3. Connect GitHub, select the repository, and use these build settings:
   - Build command: leave blank
   - Publish directory: `.`
4. Deploy the site.

## Deploy to Netlify Drop
Drag the entire `rn-concierge-real-site` folder or the ZIP file into Netlify Drop.

## Update content
Edit the HTML files directly. Images are in `/assets`.
